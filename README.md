# Phosphor UI

Terminal-inspired UI for the web, built for Svelte.

Phosphor is an opinionated UI system for developer-facing applications:
dashboards, generators, tools, and internal apps.

- Built with Svelte 5
- Powered by Bits UI
- MIT licensed
- Copy-paste components (no npm dependency)

> ⚠️ Early stage. APIs and styles may change frequently.
